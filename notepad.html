Usage Instructions

Notepad‑JS is a fully featured local note editor. Use the File menu to:

  • New Note: Clears the current note after confirmation (unsaved changes are lost).
  • Open Note: Load a text file from your device into the editor.
  • Save Note: Saves the current note using the existing file name; if none exists, you will be prompted.
  • Save As: Prompts for a file name and downloads your note as a plain text file.

You can also change the text style (font family, size, and color) using the toolbar above. All operations are performed locally without external APIs.
